오류 종류 :
1. 문법적 오류 syntax error
2. 실행시에 발생되는 오류 Runtime Exception
오류가 발생하면 프로그램 수행은 중단됨
Exception은 가벼운, 논리적 오류로서 프로그램적으로 보완하면 오류가 발생되지 않고 프로그램을 계속적으로 수행시킬 수 있음=> 예외처리
오류 발생이 예측되면 오류가 발생하지 않도록 사전에 처리 : if 조건
try ~ catch ~ finally 구문 블럭을 사용해서 예외 처리
try {
//예외 발생 가능성이 존재하는 문장;
}catch(exception ) {
//예외처리 문장
}finally {
//무조건 수행, Resource 정리
}
예외를 발생시킬 때 => throw 예외타입 객체;
