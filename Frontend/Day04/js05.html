Event Model 1 방식은 이벤트 소스와 이벤트 핸들러를 연결할 때 on이벤트타입 = 핸들러
Event Model 2 방식은 w3c 표준방식으로 addEventListener(이벤트타입, 핸들러, 이벤트캡처여부)
Event Model 1 방식은 하나의 이벤트에 연결할 수 있는 핸들러가 1개만 가능
Event Model 2 방식은 하나의 이벤트에 연결할 수 있는 핸들러가 여러개 연결 가능
