javascript 함수
함수 - 특정 기능을 수행하는 코드들의 묶음(모듈화)에 이름을 선언하고 필요할때마다 호출해서 사용(재사용성)
선언적 함수 - function 이름(매개변수) { .... }
익명함수 : ( function (매개변수) { .... }).() => 즉시실행함수
var 변수 = function (매개변수) { .... }
변수()
javascript엔진은 함수 definition은 메모리(Global 영역) 먼저 로딩함
변수는 선언과 초기화 된 후에 변수를 사용할 수 있습니다.(선언과 초기화 없이 사용하면 오류)
함수적 프로그래밍의 특성 : 함수를 1급 객체로 취급하기 때문에 함수 정의를 변수에 저장할 수 있습니다.
함수는 리턴값이 있을 수도 있고, 없을수도 있습니다.
